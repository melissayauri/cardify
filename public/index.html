Cardify
  • Descárgalo
  • Descubre el código
  • Instalar

1. Descarga el archivo index.js, en el cual encontrarás el plugin y enlazálo

<script type="text/javascript" src="index.js"></script>

2. También instala la herramienta jquery

<scriptsrc="https://ajax.googleapis .com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>

3. Y finalmente tienes que ejecutar la función cardify mediante el contenedor de la imágenes.Recuerda que #container-img lo puedes cambiar como desees, solo es un nombre para el contenedor de imágenes

$(document).ready(function() { $('#container-img').cardify(); });
